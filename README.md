# sunlight-hours frontend

## Project setup
```
npm install
```
### Setup local env
```
rename example.env.local in .env.local

check if backend port is correct
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
